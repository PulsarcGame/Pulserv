PulseArc ~ An Omnidirectional Rhythm Game

Due to the slow development and lack of bugfixes for the game Intralism, we have decided to keep the game we love alive... PulseArc is an open source community driven project that is always looking for more help.

Skills we need

  • Python Developers with experience working with REST, the requests library, flask, and mongoengine.
  • Python Developers with experience in command line applications, we use the library 'click' for PulseMan, a package manager for beatmaps
  • C# Developers with experience in games design
  • C# Developers with understanding of authentication, security, and web requests/REST APIs
